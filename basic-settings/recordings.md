## 콤마  화면 녹화 방법

* 주행 중, 콤마 화면 우측 하단에 회색 동그라미 버튼 클릭하면 **녹화 시작**
* 회색 동그라미 버튼이 빨간색, 회색 번갈아가며 점등되는 상태이면 녹화중
* 다시 클릭하면, **농화 중지**&#x20;

## 녹화 영상 다운로드 방법

* Fleet 접속&#x20;
* View Screen Recordings 클릭
* 원하는 영상을 찾아, 다운로드 클릭
